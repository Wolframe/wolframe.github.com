static bool _Wolframe::protocol::Parser::getLine ( IteratorType &  src,
IteratorType &  end,
BufferType &  buf 
)
inlinestatic

Parse the current line and write its contents to a buffer Go to the end of line and write it to buffer, starting with the first non space. After call the iterator is pointing to the end of line char or at the EOF char.

Remarks
End of lines or trailing spaces are not implicitely consumed because this leads to intermediate states that cannot be handled
Control characters are mapped to space (ascii 32) before writing them to 'buf'
Parameters
[in,out]srcinput iterator
[in]endinput iterator marking the end of input
[in,out]bufoutput buffer implementing the STL back insertion sequence interface
Returns
true, if the line was consumed completely
