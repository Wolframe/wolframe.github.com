Detailed Description

template<class Buffer>
class _Wolframe::protocol::CArgBuffer< Buffer >

Buffer for multi argument parsing (fixed array of null terminated byte character strings)

Template Parameters
Bufferbuffer type with an interface as _Wolframe::protocol::Buffer, a subset of the std::string interface used for buffering the elements of the argument vector.
