In this chapter we describe how Wolframe functions can be called from command line. The programs introduced here are installed as executables together with the wolframe application server.

6.1. The program wolfilter

The program wolfilter allows you to call any Wolframe function or filter on command line. The idea is to provide a tool that allows testing of Wolframe functions without the client server context.
